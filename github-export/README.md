# Canelli Construction Website

A premium, luxury construction company website with two implementations:

## Versions Available

### 1. Vanilla HTML/CSS/JS (`/vanilla`)
- Zero dependencies (except CDN fonts/icons)
- Just open `index.html` in browser
- Perfect for simple hosting

### 2. React + Vite + Tailwind (`/react`)
- Modern component-based architecture
- Hot reload development
- Optimized production builds

## Quick Start

### Vanilla Version
```bash
cd vanilla
# Open index.html in browser
# Or use a local server:
python -m http.server 8000
```

### React Version
```bash
cd react
npm install
npm run dev
# Open http://localhost:5173
```

## Deployment

Both versions can be deployed to:
- **Vercel** (recommended for React)
- **Netlify**
- **GitHub Pages**
- **Any static hosting**

## Features

- Premium dark theme with gold accents
- 13 sections (Hero, About, Services, Portfolio, etc.)
- Interactive cost estimator
- Filterable portfolio gallery
- Contact form
- Google Maps integration
- Fully responsive
- SEO optimized
